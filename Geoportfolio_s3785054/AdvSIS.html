Advanced Spatial Information Science
Advanced Spatial Information Science (SIS) involved broadening one's knowledge in using other Geographical Information System (GIS) software aside from the common and industry standard but also expensive ArcGIS suite.

The course involved learning a range of other map making software such as the free GIS software QGIS, creating a map in CARTO and learning about different stages when it comes to publishing data onto the web, specifically about back-end databases (PostGIS/pgAdmin 4), middleware (GeoServer) and front-end software (e.g. CARTO).

The most significant component of the course was the semester-long major project of using open-source data and programs to create an output along with a report of the capabilities of using publicly available data and software. My project was creating a simplified Digital Twin of Honiara's built and natural environment using data of Honiara that was available online. The prorgrams and functions used to generate the 3D model is illustrated in the flow chart at the bottom of the page.
