Distributed Mapping
Distributed Mapping is about using websites and web programming in conjunction with displaying maps. The course used HTMl, CSS and a bit of JavaScript to create usable websites that display maps created on online platforms such as MapBox, Leaflet and even ArcGIS Dashboards. This subject provided a very strong basis for content learned in Advanced SIS along with creating this page!

My project was creating maps that visualise the Urban Heat Island (UHI) effect and tree cover within Greater Melbourne. I have also created a map that showcases tourist areas within Melbourne too. Unlike the other gallery pages (except for Advanced SIS), the projects will be contained in individual pages that due to maps needing some time and processing power to load, so having all of the maps on the one page might cause the page to crash for weaker computers.

Note: The ArcGIS Dashboard may require an ArcGIS account to access the full functions on.
